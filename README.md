# Object ID Generator

## Installation

You can install the package via [Composer](https://getcomposer.org).

```bash
composer require orz14/object-id
```

## Usage

```php
use ObjectId\ObjectId;

$objectId = ObjectId::generate();
echo $objectId; // 666f346400408b392ccd3a35
```
